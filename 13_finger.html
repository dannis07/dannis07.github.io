Enhancing Hash Function using fingerprint

This research was carried out to increase cryptographic security by enhancing the hash function. After realizing that in the hash function, the same input results in the same output, the fingerprint was utilized as a key, thus leading to a different output. It not only took more time to Brute Force the new program but also developed tolerance to other hacking tools.
